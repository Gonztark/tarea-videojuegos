Se puede decir que los videojuegos son piezas de software creadas para entretenerse. Estos implican la interacción con una interfaz de usuario o un dispositivo de entrada (como un teclado, un controlador o un dispositivo sensor de movimiento) para generar retroalimentación visual para un jugador. Esta retroalimentación a menudo se muestra en una pantalla, pero también se puede proyectar en espacios físicos mediante realidad aumentada (AR).
